---
title: 'Get Level'
openapi: 'GET /level/{level_id}'
---

### Description
Returns the data of a specific level by its ID.

### Path Parameters
- **level_id** (required): The ID of the level (integer).

### Responses
- **200**: Level response
  ```json
  {
    "id": 1,
    "title": "Level Title",
    "builder": 123,
    "thumbnail": 456,
    "plays": 789,
    "featured": true
  }
